# Terrarium Modernization Blog

This directory contains the running blog documentation of the .NET Terrarium modernization project — bringing a beloved .NET Framework 1.0 showcase application to .NET 10 as a cross-platform web app.

## Structure

```
docs/blog/
├── README.md              ← You are here
├── announcement.md        ← The Hanselman-ready final post (living draft)
└── journal/
    ├── 00-project-kickoff.md
    ├── 03-first-pixels.md
    ├── 04-tick-loop.md
    ├── 07-signalr.md
    ├── 08-canvas-rendering.md
    ├── 09-it-lives.md
    └── ...
```

## Editorial Approach

- **Audience:** .NET developers, especially those who remember Terrarium from the early .NET days
- **Voice:** Developer-to-developer. Technical but narrative. Nostalgic but forward-looking.
- **The Hanselman Test:** Would Scott publish this? If not, it's not done.
- **Blog IS a deliverable** — not an afterthought, not documentation-we'll-get-to-later

## Content Types

### Sprint Journal Entries
Per-sprint (or per-milestone) entries that capture:
- What was built
- Key technical decisions and WHY
- Before/after code comparisons (legacy → modern)
- Challenges and solutions
- Screenshots of progress

### The Announcement Post
The final, comprehensive blog post. The one that says "We did it. .NET Terrarium is back."
Maintained as a living draft in `announcement.md`, updated each sprint.
